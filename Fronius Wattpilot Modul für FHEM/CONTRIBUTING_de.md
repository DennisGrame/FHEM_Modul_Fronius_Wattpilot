# Mitwirken (Contributing)

Wir freuen uns über Beiträge zu diesem Projekt!

## Wie man beiträgt

1. **Forken** Sie das Repository.
2. Erstellen Sie einen **Feature Branch** (`git checkout -b feature/TollesFeature`).
3. Committen Sie Ihre Änderungen (`git commit -m 'Füge TollesFeature hinzu'`).
4. Pushen Sie den Branch (`git push origin feature/TollesFeature`).
5. Öffnen Sie einen **Pull Request**.

## Fehler melden

Wenn Sie einen Fehler finden, öffnen Sie bitte ein Issue mit:

- Einer Beschreibung des Fehlers.
- Schritten zum Reproduzieren.
- FHEM Log-Ausgaben (falls relevant).
